If ‘A’ means something even though it is different from itself, this is because it connects to different pieces of meaning in a a given situation. Like we said, ‘A’ is ‘B’ is ‘C’ and so on. In math, sets are groups that, even if they are endless, have an inside and an outside since they are defined by a precise rule, unlike pieces of meaning. Meaning is like a set but also not like a set: there are rules but they are not precise. ‘A’ means one thing here and now but another thing there and then. Paradoxically, ‘A’ is itself but is also not itself. The rule that governs this weird 2-dimensional land we live in, call it ‘k’, is actually not in the two-dimensional world. The rules of language cannot be spoken: you cannot say something and create the situation in which it is said at the same time. k, which is like the rule that governs how much ‘A’ is either different or has stayed the same is never here, but is in the future. If we call this difference ‘d’, then we can say that k=d because even the meaning of constant change is always in the future. And since for something to be different from itself it must at some point have been itself, d is a function of sameness, which we can call ‘r’
