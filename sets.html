Meaning is like a set but also not like a set: there are only loose rules for what is inside and what is outside and the rules change. ‘A’ means one thing here and now but another thing there and then. Paradoxically, ‘A’ is itself but is also not itself. The rule that governs this weird 2-dimensional land we live in, call it ‘k’, is actually not in the two-dimensional world. k, which is like the rule that governs how much ‘A’ is either different or has stayed the same is never here, but is in the future. If we call this difference ‘d’, then we can say that k=d because even the meaning of change is always in the future. And since for something to be different from itself it must at some point have been itself, d is a function of sameness, which we can call ‘r’
